Цель: Определить медианную стоимость квартиры  
Для оценки качества модели используются метрики RMSE, MAE и R2.

### Навыки и инструменты:

Python  
Pandas  
Scikit-learn  
PySpark  

#### Вывод:  
Проведено сравнение моделей на предобработанных данных и на начальных
